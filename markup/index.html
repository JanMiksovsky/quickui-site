QuickUI markup is a simple HTML-like file format that allows you to package the markup, CSS, and JavaScript for a control into a single file.

The use of this markup format is optional. Its primary benefits are that it:

  • Reduces the numbers of files you need to manage.
  • Keeps related markup, CSS, and JavaScript close together â€” just as with a traditional HTML document.
  • May feel more familiar to web designers comfortable working in HTML.
