Rather than creating a new inheritance scheme, QuickUI uses the inheritance system built into jQuery. All QuickUI controls inherit from a base Control class, which in turn inherits from the base jQuery class.

Because all control classes ultimately derive from the base jQuery class, all QuickUI control instances are also jQuery instances. You can show and hide control instances with $.show() and $.hide(), you can manipulate a controlâ€™s CSS styling with $.css(), you can animate a control with $.animate(), and so on.
