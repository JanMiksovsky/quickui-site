Framework overview
A summary of the features in the QuickUI framework intended for people familiar with JavaScript.
$.control() plugin
The main jQuery plugin for creating or accessing a QuickUI control from a plain jQuery instance.
Control class
The base Control class from which all QuickUI control classes inherit. This is a subclass of jQuery, so all controls also inherit the standard jQuery methods.
Guidelines
Guidelines for creating control classes that can be readily understood and reused by other developers.
