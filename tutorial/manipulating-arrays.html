Core idea: A control is a jQuery object, which means it’s an array that can contain multiple DOM elements. You can exploit this to manipulate a collection of controls in a single call.

The demo code creates a text box whose content is used as the label of a button; changes to the text are immediately reflected in the button’s label. The $buttons declaration produces an array of class BasicButton; the array will contain an item for the top element of each button in the demo. (Initially, the array will contain just one item.) The subsequent call to $buttons.content() sets the content of any buttons in one step.

Your goal: Add some more BasicButton.create() calls to create more buttons, then try typing in the text box to change the labels for all the buttons at once.
