Core idea: You can reference the QuickUI runtime and catalog files hosted on the QuickUI site, or host those files yourself.

If you want to use QuickUI in your own web project, you’ll need to include a reference to a recent version of the jQuery library, as well as the core quickui.js runtime. For convenience, you can reference the latest public copy of quickui.js hosted on this site: 

<script type="text/javascript" src="http://code.jquery.com/jquery-1.7.2.min.js"></script>
<script type="text/javascript" src="http://quickui.org/release/quickui.js"></script>

If you want to use the QuickUI Catalog controls shown in this tutorial, you’ll also need to reference the catalog's CSS and JS files: 

<link rel="stylesheet" type="text/css" href="http://quickui.org/release/quickui.catalog.css"/>
<script type="text/javascript" src="http://quickui.org/release/quickui.catalog.js"></script>

You can also download these files from this site and host them on your own site. You may also want to clone the QuickUI repository on GitHub.
